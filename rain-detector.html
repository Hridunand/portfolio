Automatic Rain Detector and Cloth Protection System

This project is designed to automatically detect rain using a rain sensor and protect clothes that are being dried outdoors. The core components include an Arduino Uno, a rain sensor module, and a servo motor.

When rain is detected, the system automatically pulls a protective cover over the clothes, offering a hands-free solution for unpredictable weather.

Arduino Code


#include <Servo.h>

Servo tap_servo;
int sensor_pin = 4;
int tap_servo_pin = 5;
int val;

void setup() {
  pinMode(sensor_pin, INPUT);
  tap_servo.attach(tap_servo_pin);
}

void loop() {
  val = digitalRead(sensor_pin);
  if (val == 0) {
    tap_servo.write(0);
  }
  if (val == 1) {
    tap_servo.write(180);
  }
}
